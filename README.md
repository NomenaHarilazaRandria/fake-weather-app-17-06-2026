# Fake Weather App

A simple weather application built with HTML, CSS, and JavaScript.

This project simulates a weather API request using Promises, `async/await`, and `setTimeout`. It displays weather information after a short delay and handles both successful and failed requests.

## Features

- Simulated API request
- Loading state display
- Async/Await usage
- Promise handling
- Error handling with `try/catch`
- Dynamic DOM rendering
- Button disabling during requests
- Responsive and clean UI

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)

## How It Works

1. User clicks the **"Afficher la météo"** button.
2. A fake API request is simulated using `setTimeout`.
3. The application displays a loading message.
4. The request randomly succeeds or fails.
5. Weather data or an error message is displayed.
6. The button is re-enabled after the request completes.

## Concepts Practiced

- Promises
- `resolve()` and `reject()`
- `async` / `await`
- `try / catch / finally`
- DOM Manipulation
- Event Handling
- Dynamic Content Rendering
